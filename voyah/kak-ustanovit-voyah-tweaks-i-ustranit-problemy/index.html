Voyah FAQ

Как установить VoyahTweaks и устранить распространенные проблемы после установки?

29 мар 2024

Ответ

VoyahTweaks — это популярный набор модификаций для автомобилей Voyah, позволяющий расширить функционал мультимедийной системы. Для установки приложения необходимо:

  1. Выбрать версию под вашу прошивку:
  2. Для дорестайлинговых версий до 2023 года (с чипом NXP) используйте специальные инструкции для Windows или Mac.

  3. Для новых версий (рестайлинг 2023+) подойдут стандартные методы установки через ADB-подключение.

  4. Включить режим отладки:

  5. Активируйте «Режим разработчика» в настройках автомобиля, затем включите USB-отладку. Это обязательное условие для установки через ADB-инструменты.

  6. Установить через ADBAppControl или .bat-файл:

  7. Скачайте архив с официального ресурса или из закрепленных сообщений Telegram-чата. Запустите установку через ADBAppControl или выполните .bat-файл на ПК.

  8. Решение распространенных проблем:

  9. Не отображается интерфейс Tweaks: Перезагрузите мультимедийную систему и убедитесь, что отладка через инженерное меню отключена.
  10. Глюки системы после установки: Удалите приложение через ADB-команды (adb shell pm uninstall --user 0 ru.kachalin.voyahtweaks) или восстановите резервную копию через инженерное меню.
  11. Конфликты с другими модификациями: Совместимость с Kunbus не гарантируется. Для стабильной работы рекомендуется удалять предыдущие версии Tweaks перед установкой новых.

Если установка не прошла, проверьте актуальность прошивки и корректность выполнения инструкций. Для сложных случаев доступна платная помощь через удаленный доступ (5000 рублей в год).
